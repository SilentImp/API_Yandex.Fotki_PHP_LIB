get_by_title ( $  title = null,
$  limit = null 
)

Осуществляет поиск по коллекции альбомов с заданным заголовком.

Аргументы:
title название альбома. Обязательный аргумент.
limit максимально допустимое количество элементов выборки. Если установлено, то по достижении указанного числа найденных альбомов поиск будет завершен. В противном случае будут проверены все альбомы выборки на всех страницах. Если равно 0, то игнорируется.
Возвращает:
FALSE если альбомов с таким названием не найдено, альбом, если найдено единственное соответствие и массив альбомов, если найдено более одного вхождения.
См. также:
yandex_fotki_album
search ( $  order = "updated",
$  offset_time = null,
$  offset_id = "",
$  limit = 100,
$  token = null 
)

Выполняет поиск по коллекции с заданными условиями.

Аргументы:
order Порядок отображения элементов выдачи. Допустимые значения: updated (по умолчанию) - по времени последнего изменения, от новых к старым; rupdated - по времени последнего изменения, от старых к новым; published - по времени загрузки (для фотографии) или создания (для альбома), от новых к старым; rpublished - по времени загрузки (для фотографии) или создания (для альбома), от старых к новым;
offset_time Время создания ресурса в формате UTC с точностью до секунды.
offset_id Численный идентификатор ресурса на Яндекс.Фотках.
limit Количество элементов на странице выдачи.
token Токен, подтверждающий аутентификацию пользователя. Если не задан, используется токен, который был передан конструктору. Если не задан и он, то метод вызовет исключение.

Подробное описание

Класс, который позволяет вам управлять пользователями, от литца которых осуществляется работа с сервисом Яндекс.Фотки.

Конструктор(ы)

__construct ( $  login = null,
$  password = null 
)

Конструктор. Позволяет сразу создать первого пользователя. Если он создан, то автоматически становится текущим.

Аргументы:
login строка, содержащая логин пользователя. Необязательный аргумент. Если не указан, пользователь создан не будет.
password строка, содержащая пароль пользователя. Необязательный аргумент. Если указан логин, но не указан пароль, то пользователь будет создан, но не будет аутентифицирован.
Возвращает:
текущего пользователя, если он был создан.

password строка, содержащая пароль пользователя. Необязательный аргумент. Если не указан пароль, то пользователь будет создан, но не будет аутентифицирован.
